Project 2:Ideas: Registered users can submit links to photos and video’s which then based on user ‘likes’ would be ranked based on most likes.User Stories:1.	User can register themselves with a user ID and password2.	Users can upload photos/video links 3.	Users would be able to comment on photos/video links uploaded4.	Full CRUD functionality for both photos and comments5.	Users can vote/like photo/video links uploaded by other users.6.	Users can view all links based on most likes/votes
